# 五升六数学桥（math-bridge-5to6）

面向江苏小学生的五升六数学暑期衔接自学工具。项目目标是帮助孩子从“五年级下学期旧版教材”平稳过渡到“六年级上学期新版教材”，重点补齐教材调整可能造成的知识断层。

## 核心内容

- **核心知识思维导图**：按知识板块拆分，标注难度、掌握等级、前后关联。
- **核心概念速查表**：整本六上衔接必考概念、公式、单位、常用结论。
- **分层互动练习题**：基础题巩固概念，提高题专攻易错，挑战题拓展思维。
- **生活化解读手册**：把分数、体积、比、百分数放进真实生活场景。
- **数学避坑指南**：高频错点、错误示范、正确做法、记忆口诀。
- **本地进度记录**：浏览器 localStorage 保存孩子学习进度，无需后台。

## 第一阶段：纯静态网页版

打开方式：

```bash
# 直接双击打开
web/index.html
```

或用任意静态服务器：

```bash
cd web
python -m http.server 8000
```

然后浏览器访问：

```text
http://localhost:8000
```

## 第二阶段：迁移微信小程序

本项目的数据结构已经尽量按小程序迁移方式设计：

```text
web/js/data.js       # 知识点、公式、题库、生活案例、避坑指南
web/js/storage.js    # 本地进度记录
web/js/app.js        # 页面交互逻辑
```

后续迁移到微信小程序时，可以把 `data.js` 拆分为：

```text
miniprogram/data/knowledge.js
miniprogram/data/formulas.js
miniprogram/data/exercises.js
miniprogram/data/mistakes.js
```

## 是否需要后台 API？

第一版不需要。题目抽卡从内置题库随机抽取，进度保存在孩子当前设备本地。这样最稳定，也没有服务器成本。

后续如果需要 AI 个性化能力，可以再增加 API，例如：

- 根据错题自动生成同类变式题；
- 自动讲解孩子的错因；
- 生成家长周报；
- 根据学习记录推荐下一周计划。

## 版权说明

项目只放原创整理的知识点、学习路径、练习题和讲解，不上传教材扫描图、课本原文大段内容、教辅原题截图。

## 推荐使用方式

每天 35–50 分钟：

1. 5 分钟：看今天对应知识点；
2. 10 分钟：看速查表和例题；
3. 20 分钟：做基础题和提高题；
4. 5–10 分钟：记录错题，看避坑指南。

暑假目标不是提前学完六上，而是把“分数乘法、分数除法、长方体正方体、比、百分数”这条主线打通。
