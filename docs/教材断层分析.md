# 教材断层分析

## 目标

帮助孩子从五年级下学期旧版教材，平稳过渡到六年级上学期新版教材。暑假不追求提前刷完整本六上，而是补齐关键桥梁知识。

## 核心判断

五下旧版重点通常包括：

- 简易方程；
- 折线统计图；
- 因数和倍数；
- 分数的意义和性质；
- 分数加法和减法；
- 圆；
- 解决问题的策略。

六上新版衔接重点通常更快进入：

- 长方体和正方体；
- 分数乘法；
- 分数除法；
- 比；
- 百分数；
- 分数、比、百分数综合应用。

如果孩子五下没有系统学过“长方体正方体”和“分数乘法”，六上开学后会出现明显断层。

## 断层一：长方体和正方体

长方体和正方体是从平面图形到立体图形的升级，涉及空间想象、表面积、体积、容积和单位换算。

常见困难：

- 表面积和体积混淆；
- 无盖盒子、水池、通风管等少算面的问题；
- cm³、dm³、m³、L、mL 换算不熟；
- 不会把生活场景翻译成数学模型。

暑假目标：孩子需要能独立判断题目求的是“外面一层皮”还是“里面占多少空间”。

## 断层二：分数乘法

分数乘法是分数除法、比、百分数应用的前提。孩子要理解“求一个数的几分之几”就是乘法。

常见困难：

- 不理解单位“1”；
- 连续求几分之几时单位“1”变化；
- 只会套公式，不会画线段图；
- 先约分再计算不熟。

暑假目标：孩子需要能说清哪个量是整体，几分之几对应哪个部分。

## 断层三：分数除法和反向思维

分数除法的应用题经常是“已知一个数的几分之几是多少，求这个数”。这是从正向计算到反向推理的转变。

常见困难：

- 除以分数时把被除数也倒过来；
- 不会用线段图找单位“1”；
- 看到“是、占、相当于”不会判断谁和谁比较。

## 断层四：比与百分数

比、除法、分数、百分数是同一组关系的不同表达。六上后半段很多题不是单独考计算，而是考孩子能不能把这些表达互相转换。

## 推荐暑期顺序

1. 分数基础温故；
2. 长方体正方体；
3. 分数乘法；
4. 分数除法；
5. 比；
6. 百分数；
7. 综合应用和错题复盘。
